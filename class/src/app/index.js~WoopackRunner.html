    public plugin(woopack: Woopack) source

    This is called when Woopack is present and tries to load the plugin. It will add events listeners so every time Woopack builds a target or creates a revision file, the runner file will be updated. It also adds a listener for when Woopack copies the projects files so it will include the runner file.
